  ├── 📄 RUN-SERVER.bat          ← SUNUCU BAŞLATMA (Windows)
  ├── 📄 start-server.sh         ← SUNUCU BAŞLATMA (Mac/Linux)
  ├── 📄 server.py               ← Python API Sunucusu
  ├── 📄 models.json             ← Model Metadata

🚀 HEMEN BAŞLA:

  1️⃣  RUN-SERVER.bat dosyasına çift tıkla
  
  2️⃣  Bekleme... (3-5 saniye)
  
  3️⃣  Browser'da aç: http://localhost:8000
  
  4️⃣  "Hazır Sahne.glb" modelini seç

───────────────────────────────────────────────────

📦 YENİ MODEL EKLEME:

  1. Yeni .glb dosyasını models/ klasörüne koy
  2. Browser'da sayfayı yenile (F5)
  3. Dropdown'da otomatik görünecek

Örnek:
  models/
  ├── Hazır Sahne.glb
  ├── urun-1.glb          ← Ekledikten sonra
  └── urun-2.glb          ← Sayfayı yenile

  Python:    3.7+
